Module Sedlying__Sedlexing.Code_point

val lexeme : lybuf -> Stdlib.Uchar.t array

lexeme buf returns the lexeme as an array of code points.

The following functions return array segments where a segment is an array of Uchar.t,

val lexeme_segments : lybuf -> Stdlib.Uchar.t array list

lexeme_segments buf returns the lexeme as a list of array segments.

val lexeme_segments_arr : lybuf -> Stdlib.Uchar.t array array

lexeme_segments_arr buf returns the lexeme as an array of array segments.
